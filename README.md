# Playwright OrangeHRM Automation

This project automates the login functionality of the OrangeHRM demo website using Playwright and Pytest.

## Technologies Used
- Python
- Playwright
- Pytest

## Features
- Automated login test
- Parameterized test data
- SlowMo execution for debugging
- Trace viewer for debugging failures
- Page Object Model structure

## Project Structure

pages/       → Page object files  
tests/       → Test scripts  
test_data/   → Test data files  
utils/       → Helper functions  
reports/     → Test execution reports  

## How to Run the Tests

1. Install dependencies
pip install -r requirements.txt
2. Run tests
pytest
